技能：簡報產生 Pipeline（PPTX）

總覽

全自動 Quarto-to-PPTX 簡報產生 pipeline。從結構化規格到專業風格的 PowerPoint，完全由 AI agent 驅動。

模組 ID skill-presentation-pptx
版本 2.0.0
規格 v1.0
授權 Apache 2.0
GitHub ai-cooperation/skill-presentation-pptx

快速開始

git clone https://github.com/ai-cooperation/skill-presentation-pptx
cd skill-presentation-pptx

# 安裝依賴
pip install python-pptx lxml
# 需要：Quarto CLI

# 編譯範例
bash tools/compile_pptx.sh examples/deck-basic.qmd basic outputs deck.pptx

Pipeline 架構

使用者請求
    ↓
[規劃 Skill] → deck_spec.json
    ↓                (模板風格 + layout 計畫)
[工作 Skill] → deck.qmd
    ↓                (符合 layout 的正確 QMD 語法)
[compile_pptx.sh]
    ├─ 階段 1: build_template.py → template-{style}.pptx
    ├─ 階段 2: quarto render     → 原始 PPTX
    └─ 階段 3: postprocess.py    → 最終 PPTX（已修補）
    ↓
最終風格化 PPTX

7 種支援 Layout

模組的參考知識庫教導 AI agent 如何透過特定 QMD 語法觸發每種 PowerPoint layout：

Layout QMD 觸發方式
標題投影片 YAML front matter title/subtitle
標題與內容 ## Heading + 項目符號
雙欄內容 ## Heading + :::: {.columns}（純文字）
比較 ## Heading + :::: {.columns}（文字 + 圖片）
內容附標題 ## Heading + 文字 + 圖片（無分欄）
章節標題 # Heading（一級標題）
背景圖片 ## {background-image="..."} + &nbsp;

演示：版面展示

一個 QMD 原始檔即可產生 11 張專業風格投影片，展示 7 種不同的 PowerPoint layout — 全部由文件結構自動觸發。Roadmap 模板使用 PowerPoint 原生 a:glow XML 實現霓虹發光效果。

查看演示 → 下載 PPTX（Roadmap） 下載 PPTX（Pro）

關鍵技術創新

  1. Layout 觸發規則 — 記錄了 QMD 結構如何對應 Pandoc layout 選擇（Pandoc 未記錄的行為）
  2. 動態模板建構 — 無需手動編輯 PowerPoint；使用 python-pptx 程式化建構模板
  3. 後處理 — 修復 Pandoc 移除的屬性（垂直置中等）
  4. AI 知識庫 — 參考文件專為 AI 消費設計，而非人類教學
