簡報 Pipeline — 版面演示

這個互動演示展示 簡報產生 Pipeline（PPTX） 技能模組如何從結構化 QMD 標記自動觸發 7 種不同的 PowerPoint 版面。同一份原始檔可同時產生此 HTML 預覽和專業風格的 .pptx 檔案。

提示操作方式

使用 ← → 方向鍵或點擊上一張 / 下一張按鈕瀏覽投影片。

展示的版面

投影片 PowerPoint Layout QMD 觸發方式
1 Title and Content ## Heading + 項目符號
2 Two Content ## Heading + :::: {.columns}（純文字）
3a Comparison ## Heading + :::: {.columns}（文字 + 圖片）
3b Comparison（左圖右文） 左欄圖片，右欄文字
4a Content with Caption ## Heading + 文字 + 圖片（無分欄）
4b 標題 + 圖片 ## Heading + 僅圖片（上下配置）
4c 四象限對角圖文 對角對稱文字/圖片
5 Blank（背景圖） ## {background-image="..."} + &nbsp;
6 標題 + 背景圖 ## 標題 {background-image="..."} + 文字
7 Two Content 總結 — layout 觸發規則

模板設計：Roadmap

「Roadmap」模板靈感來自現代 SaaS 儀表板美學，使用 PowerPoint 原生 a:glow XML 實現霓虹發光效果：

  • 背景：深海軍藍漸層 #0D1B2A → #1B2838
  • 標題：珊瑚粉 #FF6B6B + 霓虹光暈
  • 欄標題：琥珀金 #E8A838
  • 強調/框：青綠 #4ECDC4 霓虹發光框
  • 內文：亮白 #F0F0F0

設計原則：線條或框，二選一不疊加 — 單欄 layout 用發光線條，多欄 layout 用霓虹框。
